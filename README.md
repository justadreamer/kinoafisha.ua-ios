# App

Current (legacy) version of the iOS is available here: https://itunes.apple.com/us/app/kinoafisha/id941362794?ls=1&mt=8 (available only in Ukrainian AppStore for now).

# Purpose

Initially this iOS app was intended as a demo of the XSLT tech for website scraping.  Recently it also served as a lab to try new SwiftUI and Combine frameworks introduced by Apple as part of iOS 13 SDK.  

# Source

http://kinoafisha.ua website as a source of cinema schedules info in Ukrainian cities.

# Build

Recently the app has been updated to target iOS 13 beta, using SwiftUI and Combine and builds with Xcode 11 beta 5.
